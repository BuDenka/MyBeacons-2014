<b>專題介紹</b><br>
Bluetooth Id 提供簡易的藍芽身分辨識功能，讓使用者可搜尋周遭附近持有藍芽設備的使用者，且安裝此APP軟體可讓初次見面的人知道使用者所公開的資料。
<hr>

<b>專題之需求</b><br>
用於社交、尋人
<hr>

<b>專題之功能</b><br>
使用 Bluetooth Id 搜尋自身周遭一定的範圍，列出使用者的公開資料。
